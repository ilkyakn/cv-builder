# CV Builder – Live Preview Resume Generator

A modern, browser-based CV / Resume Builder that allows users to create a professional resume with a real-time preview.

Users fill in a form and instantly see their CV updated side by side. The project is fully client-side and does not require any backend or installation.

---

Features

- Live form input with instant CV preview  
- Profile photo upload with default avatar support  
- CV completion progress indicator  
- Section visibility toggles (About, Education, References, Photo)  
- Multi-language ready structure  
- Download CV as PDF  
- Responsive and modern layout  
- Works locally (no server required)

How to Use

   git clone https://github.com/your-username/your-repo-name.git
